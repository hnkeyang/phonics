## st
/st/。在词首或词中，s 后的 t 往往会受影响变音（如 stop 读起来像 s-dop），而词尾的 st 发音则清晰完整。
词首/词中 (st)：
发音类似 /st/，但 t 常会浊化为类似 /d/ 的音。 示例： stop /stɒp/ (或听起来像 sdop), student /ˈstjuːdənt/
词尾 (st)：
发音完整，清晰地读出 /st/。 示例： list /lɪst/, test /test/, best /best/。
特殊情况（变复数/第三人称单数）：
以 st 结尾的单词变复数加 -s 时，st 后的 s 发 /s/，组合为 /sts/。示例： tests /tɛsts/, lists /lɪsts/。
