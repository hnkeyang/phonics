# 英语自然拼读法 (Phonics)

## 什么是自然拼读法？

自然拼读法（Phonics）是指看到一个单词，就可以根据英文字母在单词里的发音规律把这个单词读出来的一种方法。即从「字母发音-字母组合发音-单词-简单句子-整段句子」逐步学习，让孩子轻松掌握「自然拼读」法，培养孩子正确的英语语感，打好英语学习的基础。

在英语母语国家的幼儿园里，孩子们从三岁起，就开始接受自然拼读法的学习了，这种方法是美国及很多英语母语国家孩子学习自己母语的方法。

英语是拼音文字，所有的词汇都是由26个字母组成的，而在成千上万的不同拼法中，基本发音音素却只有48个，这说明26个字母与基本音素间是有着一定关联的。尽管如此，人们还是总结出了很多发音规律，这些规律对于绝大多数英文词汇都是适用的。

---

## 自然拼读六阶成功法

| 阶段 | 目标 | 示例 |
|------|------|------|
| 第一阶段 | 建立字母与字母自然发音之间的直接联系 | A-Z 字母发音 |
| 第二阶段 | 能够成功拼读元音+辅音（辅音+元音） | c-a → ca, a-t → at |
| 第三阶段 | 能够成功拼读辅音+元音+辅音 | d-o-g → dog |
| 第四阶段 | 能够成功拼读双音节或多音节单词 | sw-ea-t-er → sweater |
| 第五阶段 | 能够听音辨字，即听到单词读音就能拼出该单词 | 听音写词 |
| 第六阶段 | 单词量大量扩充，能够阅读英语文章 | 流利阅读 |

## 第一步：26个英文字母的发音

### 重要概念：读音 (Name) vs 发音 (Sound)

- **读音 (Name)**：字母本身的名称，如 A 读作 /eɪ/，B 读作 /biː/
- **发音 (Sound)**：字母在单词中的实际发音，如 a 在 cat 中读作 /æ/

自然拼读法学习的是字母的**发音 (Sound)**，而非读音。

### 1. 21个辅音字母的发音

实际上只有 **18种** 发音：
- C/K 发音相同，只能算一个
- W、Y 是两个半元音字母

| 字母 | 发音 | 示例单词 |
|------|------|----------|
| B | /b/ | ball, book |
| C | /k/ | cat, cup |
| D | /d/ | dog, door |
| F | /f/ | fish, fan |
| G | /g/ | go, girl |
| H | /h/ | hat, house |
| J | /dʒ/ | jump, juice |
| K | /k/ | kite, king |
| L | /l/ | lion, leg |
| M | /m/ | man, moon |
| N | /n/ | nose, net |
| P | /p/ | pig, pen |
| Q | /kw/ | queen, quick |
| R | /r/ | rat, red |
| S | /s/ | sun, sit |
| T | /t/ | top, ten |
| V | /v/ | van, vet |
| W | /w/ | win, wet |
| X | /ks/ | box, fox |
| Y | /j/ | yes, yellow |
| Z | /z/ | zoo, zebra |

### 2. 5个元音字母的短元音发音

| 字母 | 发音 | 示例单词 |
|------|------|----------|
| A | /æ/ | cat, hat, map |
| E | /e/ | bed, pen, red |
| I | /ɪ/ | sit, pig, fish |
| O | /ɒ/ | dog, hot, pot |
| U | /ʌ/ | cup, bus, sun |

**短元音规则**：当单词或音节中只有一个元音且不在末尾时，该元音通常发短音。

## 第二步：元音的字母音（长元音）

5个元音构成的以下元音组合的发音，就是该元音的字母音（读音），但必须是发长音。

### 1. A 的字母音 /eɪ/

| 组合 | 规则 | 示例单词 |
|------|------|----------|
| a_e | e在结尾不发音 | cake, tape, lake, gate, plate, grape |
| ai | 出现在以 n/l 结尾的单词中 | rain, train, mail, nail, sail, tail |
| ay | 出现在单词末尾 | play, day, today, away, say, may |

### 2. E 的字母音 /iː/

| 组合 | 规则 | 示例单词 |
|------|------|----------|
| e_e | e在结尾不发音 | these, Chinese, Japanese, athlete |
| ee | | keep, tree, three, sleep, meet, seen |
| ea | | eat, sea, tea, read, meat, clean |
| e | 只有一个元音且在单词末尾 | he, she, we, me |

### 3. I 的字母音 /aɪ/

| 组合 | 规则 | 示例单词 |
|------|------|----------|
| i_e | e在结尾不发音 | ice, nine, ride, hide, side, line, bike |
| ie | 出现在单词末尾 | die, lie, tie, pie |
| igh | 出现在辅音之间且结尾为爆破音 | right, night, light, bright |
| y | 全是辅音且出现在单词末尾 | by, fly, sky, spy, cry, dry, try |

### 4. O 的字母音 /əʊ/

| 组合 | 规则 | 示例单词 |
|------|------|----------|
| o_e | e在结尾不发音 | home, nose, stone, hope, rope, rose |
| oa | 出现在辅音之间且结尾为爆破音 | soap, load, float, road, coat |
| ow | 出现在单词末尾 | know, slow, grow, throw, low, snow |

### 5. U 的字母音

U 有两个发音：/uː/ 或 /juː/

**规则一**：当元音前的字母为爆破音时，读 /juː/

| 组合 | 示例单词 |
|------|----------|
| u_e | cube, tube, cute, cure |
| ue | Tuesday, due, cue |
| ui | suit |

**规则二**：其他情况下读 /uː/

| 组合 | 示例单词 |
|------|----------|
| u_e | rule, flute, June, rude |
| ue | blue, true, glue, clue |
| ui | juice, fruit, bruise |

## 第三步：单词的自然拼读规则

### 拼读步骤

1. **找元音**：先找元音及元音组合
   - 如单词 say、warmer、unhappy

2. **确定音节**：有几个元音或元音组合就有几个音节
   - say: 1个音节
   - warmer: 2个音节
   - unhappy: 3个音节

3. **划分音节**：
   - 两个元音或元音组合之间只有一个辅音，辅音归后
   - 两个元音或元音组合之间有两个辅音，前后各一个

4. **按音节拼读**：元音或元音组合 + 最近的辅音 + 次辅音

### 示例

**say** (1个音节)
- 第一步：写出元音组合 /ay/
- 第二步：加上辅音 /s/
- 拼读：ay → say

**warmer** (2个音节: war-mer)
- 第一个音节：ar → war
- 第二个音节：er → mer
- 组合：war + mer → warmer

**customer** (3个音节: cus-to-mer)
- 第一个音节：u → cu → cus
- 第二个音节：o → to
- 第三个音节：er → mer
- 组合：cus + to + mer → customer

## 第四步：辅音的混合

### 1. 双辅音 (Double Consonants)

双辅音在单词中只发一个音（短音节）

| 双辅音 | 发音 | 示例单词 |
|--------|------|----------|
| gg | /g/ | egg |
| dd | /d/ | odd |
| ss | /s/ | glass |
| pp | /p/ | apple |
| zz | /z/ | jazz |
| ll | /l/ | call, bell |
| nn | /n/ | ten |

**特殊情况**：元音后面的 /l/ 和 /n/，只发音后半部分

### 2. 混合辅音 (Consonant Blends)

混合辅音在单词中只发一个音（短音节）

**重要的混合辅音组合**：

| 组合 | 示例单词 |
|------|----------|
| br/bl | break, brush, branch, blue, black |
| cr/cl | cry, crab, clean, clock |
| dr | drive, dress, drink |
| fr/fl | frog, fruit, fly, flower |
| gr/gl | green, grape, glass, glad |
| pr/pl | present, prince, play, plane |
| tr | tree, train, truck |
| sc/sk/sl/sm/sn/sp/st/sw | school, sky, sleep, small, snake, spoon, star, swim |

### 3. 辅音连缀

重要的辅音连缀为 5对、7组 /s/：
- /br/, /bl/, /cr/, /cl/, /dr/, /fr/, /fl/, /gr/, /gl/, /pr/, /pl/, /tr/

## 第五步：特殊发音组合

### 1. 字母 C/G/Y 的特殊发音

#### Soft C / Hard C

| 类型 | 条件 | 发音 | 示例单词 |
|------|------|------|----------|
| Soft C | C 出现在 i, y, e 之前 | /s/ | city, cycle, face, rice, circle, nice |
| Hard C | 其他情况 | /k/ | cat, cup, cut |

#### Soft G / Hard G

| 类型 | 条件 | 发音 | 示例单词 |
|------|------|------|----------|
| Soft G | G 出现在 i, y, e 之前 | /dʒ/ | cage, village, orange, gym |
| Hard G | 其他情况 | /g/ | go, get, dog |

### 2. OO 组合的发音

## 第六步：连读规则

### 1. 爆破音、摩擦音与转音规则

#### 爆破音 (Plosives)

| 清辅音 | 浊辅音 |
|--------|--------|
| p | b |
| t | d |
| k | g |

**特点**：发音器官在口腔中形成阻碍，然后气流冲破阻碍而发出的音
- 清辅音：有气流、无声音（声带无振动）
- 浊辅音：有气流、有声音（声带振动）

#### 摩擦音 (Fricatives)

| 清辅音 | 浊辅音 |
|--------|--------|
| s | z |
| f | v |
| th (无舌) | th (有舌) |

**特点**：牙齿合拢，气流从牙齿缝中发出来
- 清辅音：牙齿中间无舌头，如 bath, mouth, thing
- 浊辅音：牙齿中间有舌头，如 the, this, that

#### 转音规则

摩擦音 s 与爆破清辅音 p、t、k 结合，转读爆破浊辅音 b、d、g：

| 拼写 | 发音 | 示例单词 |
|------|------|----------|
| sp | /sb/ | spy, spill, spoon, speak |
| st | /sd/ | stay, still, stone, student |
| sk | /sg/ | sky, skill, school, skate |

### 2. 不完全爆破

朗读时，六个爆破音在某些情况下不必爆发，即气流不需要突破阻碍，只是发音器官在口腔内形成障碍，停顿后立即过渡到后面的音。

#### 单词内部

如：bla**(ck)**board, foo**(t)**ball, ke**(p)**t
括号中的辅音只作短暂停顿，不发出声音。

#### 相邻单词之间

1. **爆破音 + 爆破音**：前一个爆破音失去爆破
   - a bi**(g)** car

2. **爆破音 + 摩擦音** (/f/, /s/, /th/, /ch/)：前一个爆破音失去爆破
   - I didn'(t) say so.

3. **爆破音 + /t/, /d/, /tr/, /dr/**：前一个爆破音失去爆破
   - a grea**(t)** change

4. **爆破音 + /m/, /n/, /l/**：前一个爆破音失去爆破
   - a bi**(t)** more

### 3. 连读 (Liaison)

在同一意群中，相邻的词如果前一个词以辅音结尾，后一个词以元音开头，可以连读。

## 学习建议

1. **循序渐进**：按照六个阶段逐步学习，不要急于求成
2. **大量练习**：通过反复拼读单词来巩固发音规则
3. **大量阅读**：掌握基本规则后，需要通过大量阅读来增加词汇量
4. **培养语感**：自然拼读法的最终目标是培养正确的英语语感

在美国及很多英语母语国家，孩子们在幼儿园都首先要掌握自然拼读法，在小学三年级前要把基本法则全部掌握，并成为熟练的阅读者。掌握了自然拼读法的基本规律后，学生们就需要靠大量的阅读来增加词汇量，丰富自己头脑里字母与发音的对应关系。
